Sobre este Proyecto

Esta aplicación web unifica todos los proyectos desarrollados en el Laboratorio de Lenguaje en una sola interfaz. Cada proyecto se puede acceder a través del menú de navegación, manteniendo una URL constante.

Los proyectos abarcan diferentes tecnologías y conceptos, desde HTML básico hasta aplicaciones con almacenamiento de datos y visualizaciones gráficas.
