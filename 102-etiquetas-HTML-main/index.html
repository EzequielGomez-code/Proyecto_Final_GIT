Etiqueta Descripción Ejemplo Elemento Adicional
<!DOCTYPE> Define el tipo de documento HTML <!DOCTYPE html> Debe ser la primera línea
<html> Elemento raíz del documento <html>...</html> Contiene todo el contenido HTML
<head> Contiene metadatos y enlaces a recursos <head>...</head> No se muestra en la página
<meta> Proporciona metadatos sobre el documento <meta charset="UTF-8"> Se coloca en el head
<title> Define el título de la página <title>Mi Página</title> Aparece en la pestaña del navegador
<body> Contiene el contenido visible de la página <body>...</body> Se muestra al usuario
<header> Representa el encabezado de una sección o página <header>Encabezado</header> Utilizado para identificar la cabecera
<nav> Define una sección de navegación <nav><a href="#">Inicio</a></nav> Contiene enlaces de navegación
<main> Contiene el contenido principal del documento <main>Contenido principal</main> Debe ser único por página
<footer> Define el pie de página <footer>© 2024</footer> Información de contacto o derechos
<section> Define una sección en el documento <section>Sección de contenido</section> Agrupa contenido relacionado
<article> Representa un contenido independiente <article>Artículo</article> Contenido autónomo
<aside> Contenido secundario o complementario <aside>Información adicional</aside> Usado en barras laterales
<h1> Encabezado de nivel 1, el más importante <h1>Título Principal</h1> Generalmente se usa una vez por página
<h2> Encabezado de nivel 2 <h2>Subtítulo</h2> Para subsecciones
<h3> Encabezado de nivel 3 <h3>Encabezado 3</h3> Subsecciones menores
<h4> Encabezado de nivel 4 <h4>Encabezado 4</h4> Para secciones internas
<h5> Encabezado de nivel 5 <h5>Encabezado 5</h5> Menor importancia
<h6> Encabezado de nivel 6, el de menor importancia <h6>Encabezado 6</h6> Para sub-subsecciones
<p> Define un párrafo de texto <p>Este es un párrafo.</p> Agrupa texto en bloques
<br> Inserta un salto de línea Texto
Otra línea		 No requiere etiqueta de cierre
<hr> Inserta una línea horizontal <hr> Separa secciones
<blockquote> Cita un bloque de texto <blockquote>Cita textual</blockquote> Generalmente con sangría
<ol> Define una lista ordenada <ol><li>Item 1</li></ol> Usa números o letras
<ul> Define una lista desordenada <ul><li>Item</li></ul> Utiliza viñetas
<li> Elemento de una lista <li>Elemento</li> Dentro de <ul> o <ol>
<dl> Define una lista de definiciones <dl><dt>Término</dt><dd>Definición</dd></dl> Agrupa términos y definiciones
<dt> Define un término en una lista de definiciones <dt>HTML</dt> Usado dentro de <dl>
<dd> Define la descripción de un término <dd>Lenguaje de marcado</dd> Complementa a <dt>
<figure> Agrupa contenido multimedia <figure> <img src="img.jpg" alt="Imagen"> <figcaption>Leyenda</figcaption> </figure> Usado para imágenes y leyendas
<figcaption> Define la leyenda de un elemento <figure> <figcaption>Descripción de la imagen</figcaption> Primero o último hijo de <figure>
<div> Contenedor o división genérica <div>Contenido</div> Utilizado para agrupar elementos
<a> Define un hipervínculo <a href="https://example.com">Enlace</a> Usa el atributo href
<img> Inserta una imagen <img src="imagen.jpg" alt="Descripción"> El atributo alt es obligatorio
<em> Texto con énfasis (cursiva) <em>Importante</em> Semántico para resaltar
<strong> Texto de fuerte importancia (negrita) <strong>Resaltado</strong> Semántico para énfasis
<small> Texto en tamaño reducido <small>Nota</small> Información secundaria
<cite> Cita una obra <cite>Nombre de la obra</cite> Utilizado en referencias
<q> Cita corta en línea <q>Texto citado</q> Se muestra entre comillas
<abbr> Abreviatura o acrónimo <abbr title="HyperText Markup Language">HTML</abbr> El atributo title ofrece la definición completa
<data> Asocia un valor a una cadena legible <data value="123">Precio</data> Útil para datos estructurados
<time> Representa una fecha u hora <time datetime="2024-02-12">12 de febrero de 2024</time> Usa el atributo datetime
<code> Muestra fragmentos de código <code>var x = 10;</code> Para ejemplos de programación
<pre> Preserva espacios y saltos de línea <pre> Código formateado</pre> Ideal para mostrar código
<samp> Muestra la salida de un programa <samp>Error</samp> Indica salida del sistema
<kbd> Representa la entrada desde el teclado <kbd>Ctrl + C</kbd> Muestra combinaciones de teclas
<var> Representa una variable <var>x</var> Usado en expresiones matemáticas
<sub> Texto en subíndice H<sub>2</sub>O Utilizado en fórmulas químicas
<sup> Texto en superíndice 10<sup>2</sup> Para exponentes
<i> Texto en cursiva <i>Itálico</i> Generalmente para términos técnicos
<b> Texto en negrita <b>Negrita</b> No tiene significado semántico
<u> Texto subrayado <u>Subrayado</u> No es semántico
<mark> Resalta texto <mark>Destacado</mark> Muestra fondo por defecto (amarillo)
<ruby> Contenedor para anotaciones de pronunciación <ruby>漢 <rt>kan</rt></ruby> Utilizado en idiomas asiáticos
<rt> Texto de la anotación en un ruby <rt>pronunciación</rt> Se coloca dentro de <ruby>
<rp> Proporciona paréntesis en anotaciones ruby <rp>( </rp><rp>) </rp> Complementa a <rt>
<bdi> Aísla una parte de texto de la dirección <bdi>Texto</bdi> Útil en textos bidireccionales
<bdo> Forza la dirección del texto <bdo dir="rtl">Texto</bdo> Puede cambiar la dirección del contenido
<span> Contenedor en línea genérico <span>Texto</span> Para agrupar sin romper la línea
<ins> Indica contenido insertado <ins>Nuevo</ins> Muestra cambios en el contenido
<del> Indica contenido eliminado <del>Antiguo</del> Normalmente se muestra tachado
<table> Define una tabla de datos <table><tr><td>Dato</td></tr></table> Organiza información en filas y columnas
<caption> Define la leyenda o título de una tabla <caption>Título de la tabla</caption> Se coloca como primer hijo de <table>
<thead> Agrupa el encabezado de la tabla <thead><tr><th>Encabezado</th></tr></thead> Mejora la semántica de la tabla
<tbody> Agrupa el cuerpo de la tabla <tbody><tr><td>Dato</td></tr></tbody> Contiene las filas principales
<tfoot> Agrupa el pie de la tabla <tfoot><tr><td>Pie</td></tr></tfoot> Utilizado para totales o resúmenes
<tr> Define una fila en una tabla <tr><td>Celda</td></tr> Debe estar dentro de <table>
<th> Celda de encabezado en una tabla <th>Encabezado</th> Usado en <tr> de <thead> o <tbody>
<td> Celda estándar en una tabla <td>Dato</td> Contiene la información
<form> Define un formulario interactivo <form action="#" method="post">...</form> Se utiliza para enviar datos
<input> Elemento de entrada de datos <input type="text" placeholder="Escribe aquí"> No requiere cierre
<textarea> Área de texto multilínea <textarea>Texto</textarea> Ideal para comentarios
<button> Define un botón clickable <button>Enviar</button> Puede activar scripts
<select> Crea una lista desplegable <select><option>Opción</option></select> Agrupa opciones
<option> Define una opción en un desplegable <option value="1">Uno</option> Se usa dentro de <select>
<optgroup> Agrupa opciones relacionadas <optgroup label="Grupo"> <option>Item</option> </optgroup> Mejora la organización
<label> Etiqueta para un elemento de formulario <label for="campo">Nombre:</label> Asocia texto a un input
<fieldset> Agrupa elementos relacionados en un formulario <fieldset><legend>Datos</legend>...</fieldset> Organiza secciones del formulario
<legend> Define un título para un <fieldset> <legend>Información</legend> Proporciona contexto al grupo
<datalist> Define una lista de opciones predefinidas <input list="opciones"> <datalist id="opciones"> <option value="A"> </datalist> Mejora la entrada de datos
<output> Muestra el resultado de un cálculo o acción <output>Resultado</output> Se puede actualizar dinámicamente
<script> Define un bloque de código JavaScript <script>console.log("Hola")</script> Puede estar en head o body
<noscript> Contenido alternativo si JavaScript está deshabilitado <noscript>Activa JavaScript</noscript> Se muestra si JS no funciona
<style> Define estilos CSS internos <style>body { font-family: Arial; }</style> Se coloca en head
<link> Enlaza recursos externos (hojas de estilo, iconos) <link rel="stylesheet" href="estilos.css"> Se coloca en head
<iframe> Incrusta otro documento dentro de la página <iframe src="pagina.html"></iframe> Para contenido embebido
<embed> Incrusta contenido multimedia <embed src="archivo.pdf" type="application/pdf"> Soporta diversos formatos
<object> Incrusta recursos externos (multimedia, plugins) <object data="video.mp4"></object> Alternativa a <embed>
<param> Define parámetros para plugins incrustados <param name="autoplay" value="true"> Se usa dentro de <object>
<video> Incrusta contenido de video <video src="video.mp4" controls></video> Requiere formatos compatibles
<audio> Incrusta contenido de audio <audio src="audio.mp3" controls></audio> Admite múltiples formatos
<source> Define múltiples fuentes para medios <source src="video.mp4" type="video/mp4"> Se usa en <video> o <audio>
<track> Define subtítulos o pistas de texto para medios <track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español"> Mejora la accesibilidad
<canvas> Área para dibujo mediante scripting <canvas width="300" height="150"></canvas> Utilizado para gráficos
<map> Define un mapa de imagen <map name="mapa"><area shape="rect" coords="0,0,100,100" href="#"></map> Se asocia a un <img> con usemap
<area> Define una zona activa dentro de un mapa de imagen <area shape="circle" coords="50,50,40" href="#"> Se utiliza dentro de <map>
<svg> Define gráficos vectoriales escalables <svg width="100" height="100"><circle cx="50" cy="50" r="40" /></svg> Integrado en HTML5
<math> Incorpora notación matemática (MathML) <math><mi>x</mi></math> Soporta fórmulas matemáticas
<details> Elemento interactivo para mostrar/ocultar contenido <details> <summary>Más información</summary> Contenido oculto </details> Se usa para secciones plegables
<summary> Define un resumen o título para <details> <summary>Ver detalles</summary> Actúa como botón para mostrar contenido
<menu> Define un menú de comandos o lista de opciones <menu><li>Opción</li></menu> Puede usarse para menús contextuales
<menuitem> Representa un ítem dentro de un menú <menuitem label="Acción"> Soportado en algunos navegadores
